# 🍼 Tracker de Mamadas - Baby Feed

Uma aplicação web simples e intuitiva para acompanhar as mamadas do seu bebê, com dashboard interativo e histórico completo.

## ✨ Funcionalidades

- 📝 **Registro de Mamadas**: Adicione facilmente cada mamada com data, hora, duração e tipo de alimentação
- 🤱 **Tipo de Alimentação**: Registre tanto leite materno quanto fórmula
- 📊 **Dashboard Interativo**: Visualize estatísticas do dia em tempo real
- 📈 **Gráficos**: Distribuição visual entre leite materno e fórmula
- 📜 **Histórico Completo**: Acompanhe todas as mamadas registradas
- 💾 **Armazenamento Local**: Dados salvos no navegador (localStorage)
- 📱 **Design Responsivo**: Funciona perfeitamente em desktop, tablet e celular

## 🎯 Estatísticas em Tempo Real

- Total de mamadas no dia
- Tempo total de amamentação
- Média de duração por mamada
- Distribuição por tipo de alimentação

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Preencha o formulário com:
   - Tipo de alimentação (Leite Materno ou Fórmula)
   - Duração em minutos
   - Data e hora (preenchido automaticamente com o horário atual)
3. Clique em "Registrar Mamada"
4. Acompanhe as estatísticas no dashboard e no histórico

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação
- **CSS3**: Design moderno e responsivo com gradientes
- **JavaScript (Vanilla)**: Lógica da aplicação
- **Chart.js**: Biblioteca para gráficos interativos
- **LocalStorage**: Persistência de dados no navegador

## 📱 Compatibilidade

- Chrome/Edge (recomendado)
- Firefox
- Safari
- Navegadores móveis

## 🔒 Privacidade

Todos os dados são armazenados localmente no seu navegador. Nenhuma informação é enviada para servidores externos.
